# Coding Guidelines

## Requirements

This project uses [Node.js](https://nodejs.org/).

Get started by installing dependencies, with [npm](https://www.npmjs.com/) run

```shell
npm install
```

This will trigger the [script/prepare.js](../script/prepare.js) script that will install the _git pre-commit_ hook defined by [../script/pre-commit.sh](../script/pre-commit.sh).

Start the _dev server_ with

```shell
npm start
```

## Indentation

Use [Prettier](https://prettier.io/) to handle indentation. You can launch it with `npm run prettier`.
