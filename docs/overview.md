# Project Overview

The goal of _Design System_ project is to provide

> a flexible framework of standards, reusable components, and documentation to create a minimal and adaptable UI

## Files and folders

- [css/](../css/): stylesheets that implement the design system styling.
- [docs/](../docs/): documentation in _Markdown format_ including principles, guidelines and instructions.
- [html/](../html/): contains HTML snippets to be used for testing and demonstration of the components and styling.
